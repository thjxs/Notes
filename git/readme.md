## config
配置全局的忽略文件列表，如 *sftp* 的配置文件，*vscode* 的配置文件等
global .gitignore
```
git config --global core.excludesfile /path/to/.gitignore_global
```
## diff
对比两次提交的差异
```
git diff <commit> <commit> [--] [<path>]
```
