# RegEx

## 行定位符
^x  以 x 开头的字符串
x$  以 x 结尾的字符串

## 单词定界符
ex: an 可以匹配到 and
\ban\b 则匹配不到
\Ban\B 则相反，匹配不到 an 

## 选择字符 或
Aa|aA  表示 Aa 或者 aA 
[a-d]  表示 a b c d 其中一个

排除字符
[^1-5]  表示该字符 不在 1 ~ 5 之间
