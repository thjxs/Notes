# 非负整数
```
^\d+$
```

# 以字母，下划线开头的 8-10 位字符
```
^[\w_]\w{8,10}$
```

# 替换
```
/<p>\s?(<img.*?>)\s*<\/p>/s   <div>$1</div>    # $1 => <img.*?>
```

# ascii 匹配所有 ASCII 非可视字符
```
[^\x20-\x7E]
```

# 修正数字
* 保留小数点后两位，如果第三位不为零，也需要保留
`(\.\d\d[1-9]?)\d*`

# 顺序环视
* 将 Jeffs 替换为 Jeff's
`/\bJeff(?=s\b)/Jeff's/g`

# 匹配数字
`[0-9][0-9]*$`
